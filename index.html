What is Github?

Github is, in essence, a code storing website. A good example of this would be google drive or Microsoft Azure cloud services, but for code. Meaning that whenever you write code, build a project or want to share some kind of code-based idea, Github is your go-to website for sharing and storing that project.

The goal of this assignment is to help you understand the terminology used on the GitHub website and to help you understand what we will use the site for. Additionally you will be creating a free account with GitHub. We recommend that you use a personal email address for this, but you can use your school account if needed as well. Tasks Review the following information: Some terminology on Github: Watch: Notification options. It offers “Not watching”, “Releases only”, “Watching”, and “Ignoring”. Stars: Likes on Instagram && thumbs up on Messenger Fork: Copy and clone the repository (project) to your account. This feature is used to modify the project and it is important when you are coding in a team. The “fork” feature will be discussed in more detail later in this article. The Github website that allows you to also like, comment and star other projects from other creators like yourself. So when you fork a project, you are just getting a personal copy of that project to your own account. When you star someone else's project, you are in essence telling them you like the project they made. When you watch something, you're notified when updates or changes come to that project.
